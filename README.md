# Portfolio Template
## About

<img src="about.png" alt="Profile Photo" width="1000" />

This is a **simple, modern, and fully customizable portfolio website** built using **React.js** and **Bootstrap**.  
It is designed to help developers and creatives showcase their skills, projects, and personal information in a clean and professional way.

The entire website content is controlled through a single **JSON file**, making customization easy without changing the source code.

---

## About This Portfolio

This portfolio includes the following sections:

- **Hero Section** with dynamic typing effect  
- **About Me Section** with profile details  
- **Skills Section** to display technical expertise  
- **Projects Section** to showcase work  
- **Contact Section** for communication  

The layout is fully responsive and works smoothly on all devices.

---

## Customization

All content such as text, images, skills, and projects can be updated using the `data.json` file.  
Updating this file automatically reflects changes across the entire portfolio.

---

## Purpose

This portfolio template is created to provide a **clean, professional, and easy-to-use personal website** suitable for resumes, internships, and job applications.
